Browser-Import

Importieren Sie Passwörter direkt aus Ihrem Browser.

Chrome / Edge
  1. Öffne chrome://settings/passwords
  2. Klicke auf ⋮ neben "Gespeicherte Passwörter"
  3. Wähle "Passwörter exportieren"
  4. Speichere die CSV-Datei
Firefox
  1. Öffne about:logins
  2. Klicke auf ⋮ (Menü)
  3. Wähle "Zugangsdaten exportieren"
  4. Speichere die CSV-Datei
Opera GX
  1. Öffne opera://settings/passwords
  2. Klicke auf ⋮ neben "Gespeicherte Passwörter"
  3. Wähle "Passwörter exportieren"
  4. Speichere die CSV-Datei
